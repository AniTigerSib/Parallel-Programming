## Задание.
Необходимо взять программу заполнения квадратной матрицы случайными значениями построчно из Задания 1 и добиться ускорения ее работы за счет:
- распараллеливания работы внешнего цикла;
- распараллеливания работы внутреннего цикла;
- распараллеливания работы вложенных циклов с использованием опции `collapse`.

Для получения наилучшего эффекта во всех случаях также необходимо использовать опцию `schedule` с различными параметрами (вариантами распределения итераций по потокам).

**Ответ на задание** должен быть представлен в виде файлов:
- файлы исходного кода последовательной и каждой параллельной версии программы (один файл `.c/.cpp` или несколько файлов в архиве, должно быть указано, какой вариант реализован в каждом файле/фрагменте);
- файл краткого отчета.

В кратком отчете пояснить результаты экспериментов (полученное ускорение каждого варианта относительно последовательной версии программы).

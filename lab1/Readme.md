## Задание.
В файле представлена простая программа заполнения квадратной матрицы (вариант 1 - построчно, вариант 2 - по столбцам). Необходимо выполнить следующие исследования:
1. Обеспечить измерение и вывод времени выполнения заполнения матрицы (не времени выполнения всей программы, а только времени выполнения циклов заполнения матрицы).
2. Подобрать такой размер матрицы, который требует заметного (1-2 секунды) времени заполнения матрицы. Для этого необходимо:
  - определить доступный объем оперативной памяти;
  - определить такой размер квадратной матрицы, чтобы она полностью умещалась в оперативной памяти (около 70-80% доступного объема);
  - убедиться, что при выполнении программы для этого процесса не используется виртуальная память (при необходимости скорректировать размер матрицы).
3. Создать модификации программы:
  - заполнение матрицы в направлении по строкам и по столбцам;
  - в каждом из двух случаев реализовать заполнение нулями, последовательными значениями, случайными значениями.
4. Произвести измерение времени выполнения всех 6 вариантов в случаях:
  - включенной оптимизации кода (например, опции компилятора o2, включающую автоматическую векторизацию);
  - выключенной оптимизации кода в целом или выключенной автоматической векторизации (опции см. для используемого компилятора).

Ответ на задание должен быть представлен в виде следующих файлов:
- файлы исходного кода (один файл .c/.cpp или несколько файлов в архиве, должно быть указано, какой вариант реализован в каждом файле/фрагменте);
- файл краткого отчета.

В кратком отчете представить:
1. Характеристики системы (модель процессора, количество ядер, рабочая частота, объем оперативной памяти, тип дисковой памяти и т.п.).
2. Информация об используемом трансляторе и опциях, использованных для включения/отключения оптимизации.
3. Порядок определения размера матрицы.
4. Результаты экспериментов (в наглядном виде, предпочтительно табличном, допускающем сравнительный анализ результатов экспериментов).
5. Выводы о влиянии архитектуры и характеристик системы на время выполнения экспериментов по заполнению матрицы.
